About The Project

This project is a web application designed to predict traffic situations based on real-time vehicle count data. It leverages a machine learning model to classify traffic into four categories: Low, Normal, High, and Heavy.

Objective

The main goal is to provide a tool for traffic analysis and prediction, which can help in traffic management and planning. By inputting the number of different vehicles, users can get an instant prediction of the traffic condition.

Technology Stack

  • Backend: Flask (Python)
  • Machine Learning: Scikit-learn
  • Data Handling: Pandas
  • Frontend: HTML, Bootstrap CSS

The Model

The prediction is powered by a Random Forest Classifier, a robust ensemble learning method. It was trained on the TrafficTwoMonth.csv dataset, which contains historical data on vehicle counts and their corresponding traffic situations.
